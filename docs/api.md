# Interior Design API

## Base URL

`http://localhost:5000/api`

## Endpoints

- `GET /health` - Check whether the server is running.
- `POST /auth/login` - Demo login endpoint.
- `POST /auth/register` - Demo registration endpoint.
- `GET /designs` - Get all design items.
- `GET /designs/:id` - Get one design item.
- `GET /wishlist` - Get wishlist items.
- `POST /wishlist` - Add an item to wishlist.
- `DELETE /wishlist/:id` - Remove a wishlist item.
- `GET /bookings` - Get booking requests.
- `POST /bookings` - Create a booking request.
- `GET /reviews` - Get client reviews.
- `POST /reviews` - Create a review.
